3. 权限申请与使用说明

为实现相关功能，应用可能在不同平台申请或调用以下权限/系统能力：

  • 相机权限：用于二维码扫码、图像采集类识别场景，仅在您主动扫码时启用。
  • NFC 权限/能力：用于识别、读取、写入 NFC 标签及与设备进行近场交互。
  • 文件/相册/媒体访问权限：用于读取您主动选择的方案文件、二维码图片，或将二维码图片、方案文件导出到本地设备。
  • USB Host / USB 设备访问能力：用于识别并连接外接 USB 设备，完成串口通信和配置下载。
  • 网络权限：用于访问官网、检查更新、下载安装包以及完成必要的版本分发请求。
  • 安装未知来源应用/安装包拉起能力：仅在您主动执行应用更新安装时调用系统安装器，不会在后台静默安装。
  • 振动等系统能力：可能用于扫码或交互反馈，不用于信息收集。

4. 关于定位、Wi-Fi 与手机状态权限的特别说明

当前 Android 版本可能声明 定位、Wi-Fi、手机状态 等兼容性权限， 主要用于底层系统兼容、外设能力预留或更新安装相关场景适配。现阶段根据本应用的实际业务逻辑：

  • 我们不会基于上述权限持续获取您的精准地理位置、行动轨迹或周边 Wi-Fi 信息。
  • 我们不会主动读取您的联系人、通话记录、短信内容，也不会将手机状态信息用于用户画像、广告推荐或精准营销。
  • 如后续版本对相关权限的声明或用途发生变化，我们会同步更新本隐私政策并向您说明。

5. 本地处理与本地存储说明

佰安翼当前版本以本地处理为主。您导入的二维码图片、方案文件、NFC 标签内容、USB 通信结果、 方案参数、保护密码以及导出的二维码或方案文件，原则上均在您的设备本地读取、解析、保存或写入。

  • 应用当前不提供账号注册/登录体系，因此不会因账号体系而收集您的手机号、昵称、头像、身份证号等信息。
  • 方案数据通常保存在应用本地数据库、应用支持目录、下载目录或您主动选择的导出位置。
  • 二维码图片在您主动保存时，可能写入系统相册或下载目录。
  • 设备通信日志和运行日志默认保存在本地，主要用于您自行排查或后续技术支持协助定位问题。
  • 除您主动执行在线更新外，当前版本不会将您的方案文件、二维码内容或设备配置自动上传至我方服务器。

8. 信息存储与保护
